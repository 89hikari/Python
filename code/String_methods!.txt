
    - upper() - Возвращает строку, символы которой приведены к верхнему регистру
    - lower() - К нижнему регистру
    - swapcase() - Регистр всех символоу обращён
    - capitalize() - Первая буква прописная, все остальные строчные
    - strip() - Убирает пробелы, пустые строки, табуляцию в начале и в конце
    - replace(old, new[max]) - Вхождение строки old заменяется на new. Необязательный параметр msx устанавливает наиболее возможное кол-во замен
    - find(str, [start], [end]) - Поиск подстроки в строке. Возвращает номер первого вхождение или -1.
    - rfind(str, [start], [end]) - Поиск подстроки в строке. Возвращает номер последнего вхождение или -1.
    - index(str, [start], [end]) - Поиск подстроки в строке. номер первого вхождения или ValueError
    - rindex(str, [start], [end]) - номер последнего вхождения или ValueError
    - split(символ) - Разбиение строки по разделителю
    - isdigit() - состоит ли строка из цифр
    - isalpha() - состоит и строка из букв
    - isalnum() -  состоит ли строка из цифр или букв
    - islower() - состоит ли строка из символов в нижнем регистре
    - isupper() - состоит ли строка из символов в верхнем регистре
    - isspace() - состоит ли строка из неотображаемых символов
    - istitle() - начинаются ли слова в строке с заглавной буквы
    - startswith(str) - начинается ли строка с шаблона str
    - endswith(str) - заканчивается ли строка S шаблоном str
    - join(список) - сборка строки из списка с разделителем S
